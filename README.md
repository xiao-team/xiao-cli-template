# createVue

## 项目介绍

-   本项目基于 webpack4 搭建 vue 项目。

-   本项目属于精简版，服务 xiao-cli 脚手架，因此需足够简洁，这里是完整版[项目地址](https://github.com/zxpsuper/createVue)

-   你可以用此项目搭建属于你的项目！！

-   配置好 vue-router vuex less sass pwa prettier eslint ..

-   熬夜不易，如果您觉得我的文章或者 demo 有用，可以赏我杯咖啡(^\_−)☆

        npm start

        # 打包
        npm run build
